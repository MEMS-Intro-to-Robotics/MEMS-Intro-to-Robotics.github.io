# For Educators

This site hosts the lab manuals, setup guides, and technical references for the Introduction to Robotics and Automation course (ECE 383 / ME 555) at Duke University. The material is published openly so that students have a stable, citable reference — and so that educators at other institutions can browse, adapt, or build on it.

## What you will find here

- **Ten lab manuals** spanning environment setup, ROS 2 fundamentals, shell scripting, Python nodes, MoveIt 2 manipulation, PID control, hardware validation, and autonomous SLAM-based exploration.
- **Setup and onboarding guides** for getting students into Docker-based ROS 2 workspaces quickly.
- **Technical references** for the Kinova Gen3 Lite robot arm, MoveIt 2, and the pymoveit2 Python API.
- **Starter code** for manipulation and controls labs.

## Course-specific assumptions

Some material assumes the Duke infrastructure:

- Student VMs provisioned through [Duke VCM](https://vcm.duke.edu) with FastX remote desktop
- Docker images pulled from a local registry
- Course-specific package names and support channels

If you are adapting this for your own course, replacing these infrastructure-specific pages is the natural starting point. The lab content, references, and onboarding patterns are designed to transfer with minimal changes.

## Using and citing this material

This material is open for use and adaptation. If you reference it in a publication, the following citation format is suggested:

> Kusa, E. (2026). *MEMS Intro to Robotics: Open Lab Manuals and Technical References for Simulation-First Robotics Teaching.* Available at [https://mems-intro-to-robotics.github.io](https://mems-intro-to-robotics.github.io).

If you use or adapt this material for your own course, attribution is appreciated but not required. Questions and feedback are welcome via the [GitHub repository](https://github.com/MEMS-Intro-to-Robotics/MEMS-Intro-to-Robotics.github.io).
